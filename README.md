# Executive Board Recommendations Tracking Dashboard

## 📌 Project Overview
This project was completed as part of my internship to create a **dynamic and interactive Excel dashboard** for tracking and analyzing executive board recommendations.  
The goal was to enhance the data structure, add meaningful tracking fields, and build a professional dashboard to visualize key insights.

---

## 📂 Scope of Work

### **Part 1 – Data Structure Enhancement**
Enhanced the existing recommendations table by adding new columns to improve tracking and analysis:

| Column Name           | Type         | Values / Format                                  | Purpose |
|-----------------------|-------------|--------------------------------------------------|---------|
| **Status**            | Dropdown    | Completed / In Progress / Not Started           | Track execution progress |
| **Execution Date**    | Date        | dd/mm/yyyy                                       | Document implementation timeline |
| **Responsible Dept.** | Text        | Example: HR, IT, Finance                         | Identify responsible party |
| **Recommendation Impact** | Text/List | Description of achieved results                  | Clarify outcome |
| **Notes**             | Text        | Any follow-up notes or challenges                | Track details/issues |
| **Priority**          | Dropdown    | High / Medium / Low                              | Help prioritize |
| **Completion Rate (%)**| Percentage | 0% – 100%                                        | Estimate progress |
| **Attachments/Links** | Hyperlink   | Links to supporting documents                    | Access resources |

---

### **Part 2 – Dashboard Development**
Designed an **interactive Excel dashboard** to provide insightful visualization:

#### **Key Indicators**
- Recommendations by Status (Completed / In Progress / Not Started)
- Recommendations by Department
- Priority Distribution
- Average Completion Rate

#### **Charts Used**
- **Bar Chart** – Recommendations per Department
- **Pie Chart** – Status Distribution
- **Timeline Chart** – Execution Trends

#### **Filtering Options**
- By Department
- By Status
- By Meeting Date

#### **Printable Reporting Page**
- Executive Summary Report
- List of Delayed or High-Priority Recommendations

---

## ⚙️ Technical Features
- Used **nested IFs** for dynamic status-based calculations
- Applied **COUNTIFS** for multi-condition counts
- Built **interactive slicers** for filtering
- Professional, clean, and user-friendly layout

---

## 📊 Deliverables
1. Enhanced recommendations table with new tracking columns.
2. Fully interactive Excel dashboard with slicers, charts, and reporting page.
3. Screenshots of the final dashboard.

---

## 🛠 Tools Used
- **Microsoft Excel** (Data Validation, PivotTables, PivotCharts, Slicers, Nested IFs, COUNTIFS)
- **Data Visualization** (Bar Chart, Pie Chart, Timeline)
- **Data Cleaning & Structuring**
